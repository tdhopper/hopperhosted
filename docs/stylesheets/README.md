# Synthwave 80s Theme

This directory contains the custom CSS for the HopperHosted site's retro 80s aesthetic.

## Color Palette

- **Neon Pink:** #ff006e, #ff0080, #f72585
- **Neon Purple:** #b967ff, #7b2cbf
- **Neon Cyan:** #00d9ff, #0ff0fc
- **Neon Teal:** #06ffa5
- **Background:** #0a0118, #1a0b2e

## Features

- VHS scanline effects
- Neon glow on text and elements
- Retro grid background
- Custom scrollbar styling
- Chromatic aberration effects
- Gradient animations
- Orbitron font for headers
- Share Tech Mono for code

## Inspiration

Based on 80s synthwave aesthetics, Tomorrow Night color schemes, and VHS tape covers from the 1980s.
